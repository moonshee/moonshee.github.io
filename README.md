The IMM simulator is an educational and amusement tool for new IMM user and professionals.  It is HTML based with CSS and JS and deploys exclusively on devices which run HTML such as smart phones, laptops, PCs.  It can run on web browsers like Firefox.

The simulator allows pupils and users to learn all about injection molding, particularly plastic technology.  The machine has a modern sleek chassis with a hidden screw inside the panels, this is to help conserve energy.  The hopper is large and oversized for light-out manufacturing and 5s.  The whole system is designed for lean production and 6 sigma.

To start, the student visits the demo site https://moonshee.github.io

If the pupil already has a recipe, they can use the P button on the control panel which links to the Parameter Storage screen but that requires user login which includes iris scan and palm scan.

If this is a new project, then the settings all need to be entered manually or the user can use the USB option to upload files.  To enter the settings, simply click on Clamp >> Setup or Screw Back or one of the other portal buttons from the main control panel depending on which settings you are interested in.

To check your work, click on open on close or start or stop.  This will bring you a visualization of what the machine is doing at the moment.  Is it running?  Is it stopped?  Keep in mind, you can change the settings while the machine is stopped or running.  Some of the settings take a while to kick in, like if you increase screw temperature.  Other settings can't take place until you switch the machine on manual.

The EX button, for Export Data, uses Japanese technology and is only available in Japan at this time due to export restrictions.  The software also comes in a range of languages from Japanese to Urdu, French and German, Chinese.

Included with the software is an AI agent there to help teach IMM technology and make the user feel at home.  The software works fully offline as well, if a copy of the page is saved.  However, it is optimized for online use with very lo data useage required.

To get a graph of the data, simply click on the T, trend button.  That will lead the user to all the graphic displays and other data information available for the job running.  Login is required to retrieve the data for sake of security.  When users purchase the full suite, passwords are provided.

This suite is also used for remote IMM access in lights-out manufacturing, however, it requires significant investment in hardware on-site.  Only the largest and most successful clients can afford this option.
